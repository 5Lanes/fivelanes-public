Terminology in this vision
  • Sources — each channel has a fixed color: Email coral, Text teal, Slack blue, LinkedIn purple, Meet notes orange. Used on pills, thread stripes, and highlights.
  • Status banner — plans due today and new messages since the last pipeline refresh; links jump to the schedule rail or threads.
  • Lane — top-level life area as tabs (logo color order). Select a tab to see its tracks; the first track opens automatically. Sort by recently updated or recently added.
  • Track — same card as today's Lanes page: summary, highlights, priorities, assigned threads, and lane actions. Expand within a lane to open. Use Show → Archived to review archived tracks.
  • Threads — full conversation detail below lanes: summary label, sender meta, latest updates, their availability, next steps, and message history (email). Filter by source, assignment, or sort.
  • Collapse — chevron on tracks and threads; collapse/expand controls above each section.
  • Schedule rail — Calendar shows open slots, virtual-only windows, and meetings (filter with All / Open / Meetings). Plans lists due next steps.
  • Header — Run fivelanes (manual pipeline refresh), Settings (backend, schedule, quiet hours), and Sources (tabbed setup page per channel) in the top bar.
